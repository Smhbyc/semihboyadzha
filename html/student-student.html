“From Student to Student”

Posted On June 8, 2018

This project is created to help to individual, who would like to learn about computer architecture and design. It is based on the major contents of the computer organization. The program is based on the book “Computer Organization and Design”.

It is a student based project where an experienced student is explaining the core concepts of computer architecture using the methods and techniques learnt. Furthermore, client wanted program to touch in different perspectives of the computer organization, such as fundamental explanations and also some core binary math to be understood by the users.

The project is completed using VB.Net and Visual Studio.

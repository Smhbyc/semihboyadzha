Cloud Computing Information Website

Created On April 24, 2018

The purpose of this project is to design a website that informs people about the cloud computing which gets a larger user base every second. Also, this website aims to inform its users about the cross-site attack which may harm their cloud servers if users are not careful. Website also provides a deep look on how to calculate the cost of cloud server per pebibyte.

The project is completed using front-end design and development using Bootstrap 4.
